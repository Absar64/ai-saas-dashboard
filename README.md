# Cortex Dashboard

Cortex Dashboard is a full-stack AI-powered SaaS platform built to demonstrate production-level architecture and scalable system design.

- Secure authentication (JWT-based)
- Role-based access control
- AI tool integrations (image generation / processing APIs)
- Subscription-ready backend structure
- Usage tracking and analytics dashboard
- Modular REST API architecture
- PostgreSQL database integration
- Deployed-ready frontend and backend
